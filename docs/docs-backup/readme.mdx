The Infrahub Ops CLI provides command-line tools for operating and maintaining an Infrahub deployment:

- **[Infrahub Backup](./backup/index.mdx)** — create and restore complete, verifiable recovery points on Docker Compose or Kubernetes.
- **[Infrahub Collect](./collect/index.mdx)** — gather logs, diagnostics, and metrics into a single archive for OpsMill support.

Both binaries share the same environment detection, configuration, and self-update mechanism.

## Where to start

- New to the tools? Start with the [Get started tutorial](./tutorials/getting-started.mdx) for a full backup and restore cycle.
- Ready to install? See [Install Infrahub Backup](./backup/install.mdx) or [Install Infrahub Collect](./collect/install.mdx).
- Looking for a flag or environment variable? See the [CLI command reference](./reference/commands.mdx) and [Configuration reference](./reference/configuration.mdx).
- Keeping binaries up to date? See [Update the CLI](./self-update.mdx).
