## What is Device42?

[Device42](https://www.device42.com/) is a commercial IT infrastructure and asset management platform. It discovers and inventories physical and virtual infrastructure — servers, network devices, applications, services — and tracks the dependencies between them.

## Sync directions supported

- Device42 → Infrahub

:::info

The Device42 adapter supports only **one-way synchronization** from Device42 to Infrahub. Writing data back into Device42 is not supported.

:::
