# Creating a branch in Infrahub

## Overview

The `opsmill.infrahub.create_branch` action plugin creates a new branch in Infrahub through the Infrahub SDK. This module allows you to specify the branch name, whether to sync with Git, a description, and other optional parameters.

## Usage

```yaml
- name: Create a a branch
  gather_facts: false
  hosts: localhost

  tasks:
    - name: Create a Branch 'test'
      opsmill.infrahub.create_branch:
        name: "test"
        sync_with_git: false
        description: "This is a test branch"
```
