title: Event reference

Complete reference for all events emitted by Infrahub. Each page lists every event in that category with its type identifier, description, and payload fields.

For a conceptual explanation of the event system and how to use events with rules and actions, see [Events](../../events/overview.mdx).

## Categories

| Category | Description |
|---|---|
| [Account](./account.mdx) | Account creation, updates, and deletions |
| [Artifact](./artifact.mdx) | Artifact creation and updates |
| [Branch](./branch.mdx) | Branch creation, merges, rebases, and deletions |
| [Commit](./commit.mdx) | Git repository commit operations |
| [Group](./group.mdx) | Group membership changes |
| [Node](./node.mdx) | Node creation, updates, and deletions |
| [Proposed change](./proposed.mdx) | Proposed change lifecycle and review events |
| [Schema](./schema.mdx) | Schema migration events |
| [Validator](./validator.mdx) | Validator execution events |
